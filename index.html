Installation and setup

Plugin installation

First install WordPress (Installing WordPress) , then please log in to admin panel to install WooMeet - WooCommerce extension plugin.



There are 2 ways of installing a plugin

1. Install via WordPress Admin

2. Upload via FTP directly to your server.

 

1. Install via WordPress Admin

To install the plugin correctly, you find plugin woomeet.zip file downloaded from marketplace.

Go to WordPress admin panel -> Plugins -> Add New to upload plugin.

Upload this zip file > Click on Install Now. This will install the plugin. After installation click on "Activate" link to activate the plugin. You will find a sub-menu item created as "WooMeet Settings" in your Tools menu at bottom.

 

 

2. Upload via FTP directly to your server.

To install the plugin correctly, you MUST UNZIP woomeet.zip file downloaded from marketplace.

  1. Open your FTP client and connect to your server.
  2. Locate your extracted folder and copy the "woomeet" folder in the plugins directory of your WordPress installation. Plugin directory of default WordPress installation is your_rooth_path/wp-content/plugins/

Plugin Setup

Go to "WooMeet Settings" in Tools page and put the Zoom API credentials and your TimeZone then save.

To get the Zoom API credentials, sign in to zoom.us then visit https://marketplace.zoom.us/develop/create and create a JWT app. Inside the app you will get API Key and API Secret. Save them in a safe place and use these info in your WooMeet Settings.

Create Appointments

How to create appointments .

1. Create a product



2. Select WooMeet from Product data drop down



3. Select WooMeet Appointment from tabs and put date time and duration along with other product data and publish it.



4. If all steps followed properly, you should see your WooMeet appointment like this .
